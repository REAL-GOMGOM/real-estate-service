# 내집(My.ZIP) 기능 총람

> 기준일: 2026-07-12 · https://www.naezipkorea.com
> 스택: Next.js 16 (Turbopack, Cache Components/PPR) · Neon Postgres · Vercel · MDX
> 데이터: 국토교통부 실거래가 · 한국부동산원 · 금융감독원 · 한국은행 ECOS · NEIS

## 2. 홈 (`/`) — 대시보드형 랜딩

벤토 그리드 구성. 히어로 스탯(247개 시군구 커버리지·매일 09시 갱신·국평 라이브)은 실데이터.

- **퀵액션 칩 6종**: 실거래 조회 · 변동률 지도 · 청약 정보 · 입지 분석 · 부동산 지도 · 내집마련 도구
- **최근 실거래 카드**: 강남·서초·송파·마포·용산 탭 전환
- **특이 실거래 카드**: 신고가·급등·국평 TOP (highlights API 라이브)
- **LOCATION SCORE TOP 5**: 이달의 최우수 입지 (지역 페이지 딥링크)
- **수도권 국평 시세**: 84㎡ 최근 30일 평균 · 직전 대비 변동률
- **내집마련 계산기**: 집값·자기자본·금리·기간 → 월 상환액 (ECOS 라이브 기본금리)
- **청약 일정 · 칼럼 피드 · 뉴스** 카드 + 텔레그램 채널 CTA
- **모바일 하단 탭바** (홈/지도/청약/도구/더보기)

## 3. 부동산 분석

| 페이지 | 기능 |
|---|---|
| `/transactions` 실거래 조회 | 구 선택 + 단지명 검색(`?district=`, `?q=`), 거래 카드(평수·층·직전가 대비), 상세 모달 → **브랜드 이미지 공유 카드** 생성(매매·전월세 대칭), 스파크라인 |
| `/highlights` 오늘의 주요거래 | 신고가·급등·국평 고가 3카테고리, 표/카드 전환, 당일 공개분 기준 |
| `/chart` 시세 차트 | 단지별 가격 추이(3/6/9/12개월), 구 선택 + **단지명 전국 검색**(주요 15개 구 병렬 조회) |
| `/location-map` 부동산 지도 | 지역별 입지 점수 지도 (카카오맵, 학교 레이어 포함) |
| `/region` + `/region/[id]` 지역 분석 | **147개 지역 허브** — 지역별 히어로·점수·시세, 개별 OG 이미지 |
| `/ranking` 주간 랭킹 | 최고가·거래량 랭킹, 이미지 공유 |
| `/apt/[id]` 단지 상세 | 단지 마스터 연동 상세 페이지 |

## 4. 청약 (`/subscription`)

수도권 중심 청약 일정 — 청약 중/예정 구분, 세대수·경쟁률, 1시간 캐시. 홈 카드에도 요약 노출.

## 5. 내집마련 도구

| 페이지 | 기능 |
|---|---|
| `/loan` 대출 시뮬레이터 | **정부대출 탭**: 디딤돌·신혼 등 상품별 한도/우대금리 스택, 한국은행 기준금리 추이. **은행대출 탭**: 금감원 공시 은행별 상품 금리 비교(실시간), 예금은행 평균금리 추이. 공통: DSR·LTV·DTI 판정, 상환방식 3종(원리금균등·원금균등·체증식) + 상환 스케줄, **단지 검색 → 평형 선택 칩 + 거래 월 범위 표기**로 매매가 자동 입력 |
| `/gap-guide` 갭투자 가이드 | 매매-전세 전략 가이드 |
| `/dollar` 실질 가치 | 아파트값을 **달러·비트코인·금** 기준으로 환산 — "골드/BTC 대비 우리 아파트는 얼마나 올랐나" (연도별 시세 테이블 2011~, 환율 라이브) |

## 7. 콘텐츠

- **칼럼 `/blog`**: MDX 기반 — 자체 차트 컴포넌트 17종(라인·바·히트맵·게이지·산점도 등, 오버플로/컬러 가드 내장), 카테고리, RSS, 글별 OG 이미지 자동 생성, 시리즈(데일리 실거래 리포트)
- **뉴스 `/news`**: 부동산 뉴스 큐레이션 (썸네일·카테고리)
- **리포트 `/report`**: 일일 실거래 리포트 진입점

## 8. 운영·어드민 (`/admin`)

- 칼럼 CMS: 작성/수정/미리보기(토큰 링크)/카테고리 관리, NextAuth 로그인
- **machine API** (Bearer 인증): 봇·텔레그램에서 칼럼 draft 업로드→발행, daily-stats·apt-highs push
- `/preview/[id]`: 발행 전 검토용 토큰 미리보기

## 9. 공개 API (주요)

`/api/transactions`(+districts·highlights·rent·silv·summary) · `/api/apartments/search` · `/api/loan/*`(bank-rates·cofix·rate-history·simulate) · `/api/subscription` · `/api/price-change`(+districts) · `/api/price-trend` · `/api/ranking` · `/api/gap-analysis` · `/api/calendar` · `/api/news` · `/api/crypto` · `/api/dollar` · `/api/map/schools` · `/api/health/db`
